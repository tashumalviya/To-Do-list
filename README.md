# 📝 To-Do List App

A simple To-Do List web app built using HTML, CSS, and JavaScript to manage daily tasks.

## Features

- Add tasks  
- Mark tasks as completed  
- Delete tasks  

## Tech Stack

- HTML  
- CSS  
- JavaScript  


## Files

- index.html  
- style.css  
- script.js  


## Run

Open `index.html` in your browser.
